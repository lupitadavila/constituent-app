## Getting Started

Create an `.env` file with database url

Set up database 
```bash
npx prisma init
# then
npx prisma generate
```


### npm commands
run development server
```bash
npm run dev
```

build
```bash
npm run build
```
lint
```bash
npm run lint
```


### DB commands

Apply db migrations
```bash
npx prisma migrate dev
```
View db locally
```bash
npx prisma studio
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
